О проекте

    Наш проект ИТ-сервисы для "Цифрового университета" делиться на несколько подпроектов:

  1. Веб-версия личного кабинета
  2. Приложение для Android(личный кабинет)
  3. Приложение для IOS(личный кабинет)
  4. Политайм

1. Веб-версия личного кабинета

Сайт работает уже больше года. Его цель: полностью заменить старый личный кабинет.

2. Приложение для Android(Личный кабинет)

Мобильное устройство использует каждый студент, поэтому наш проект начал разработку удобного мобильного приложения для Android-устройств, чтобы облегчить просмотр расписания и предоставить все функции личного кабинета.

3. Приложение для IOS(Личный кабинет)

Создается с той же целью, как и приложение для Android

4. Политайм

Суть проекта заключается в создании продукта, который будет хранить расписание, задачи, добавляемые старостой, индивидуальные проекты и иную активность студентов, отображать ее в удобном виде с помощью графического интерфейса, с возможностями фильтрации, сортировки и тонкой настройки.
